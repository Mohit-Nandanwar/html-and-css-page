# HTML & CSS Webpage Project

## 📄 Project Overview
This project is a basic webpage created using **HTML** for structure and **CSS** for styling.  
It demonstrates how to separate content (HTML) from design (CSS) by linking an external stylesheet.

## 🛠 Features
- Proper HTML structure with headings, paragraphs, and footer.
- External `style.css` file for all styling (no inline styles).
- Use of **element selectors** (body, p, h1, etc.), **class selectors** (`.head`), and **ID selectors** (`#para`, `#bglime`).
- Styling applied to text, background colors, and links.
- Footer placed correctly inside the `<body>` tag.

## 🚀 How to Run
1. Download or clone the project files.
2. Keep `index.html` and `style.css` in the same folder.
3. Open `index.html` in any browser to view the styled webpage.

## 🎨 CSS Concepts Demonstrated
- Element, class, and ID selectors
- Text styling (color, font, size, alignment)
- Background styling
- Link hover effects
